搭建CTF平台

发表于 2020-01-28 更新于 2020-03-05 分类于 CTFd 阅读次数：

想搭一个CTF平台给大家刷题，顺便记录一下搭建过程：

过年有活动就在阿里云上租了一个服务器练手：

服务器：Centos 7.4 64位

开源平台：CTFd

一 安装docker

安装yum

1.更新yum包

yum update

2.安装必要的系统工具

yum install -y yum-utils device-mapper-persistent-data lvm2

3.设置yum源

yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装docker

4.查看并选择合适的docker版本

yum list docker-ce --showduplicates | sort -r

5.安装docker

yum install docker-ce-17.12.1.ce

6.DaoCloud配置docker镜像源加速

curl -sSL https://get.daocloud.io/daotools/set_mirror.sh | sh -s http://f1361db2.m.daocloud.io

安装docker-compose

1.选择最新的docker-compose版本并下载，官方源安装

sudo curl -L "https://github.com/docker/compose/releases/download/1.25.4/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

由于过于缓慢换源

curl -L https://get.daocloud.io/docker/compose/releases/download/1.25.0/docker-compose-`uname -s`-`uname -m` > /usr/local/bin/docker-compose

2.设置docker-compose执行权限

sudo chmod +x /usr/local/bin/docker-compose

docker-compose --version

二 安装CTFd

安装git

sudo yum install -y git

从Github上克隆CTFd

git clone https://github.com/ctfd/ctfd.git

进入ctfd文件夹，使用 docker-compose 启动项目

cd CTFddocker-compose up -d

静静等待安装完成，成功会弹出http://0.0.0.0:8000,此时，你的ctfd就安装完成了,
如果安装成功不能访问，看下是不是防火墙把端口给过滤掉了。
